# sample workspace

Anything in this directory is **owned by the agent**: it may be read,
grepped, written, or edited by `/api/agent`. Files outside this directory
are out of reach (the `Tools` class enforces a path-traversal check).

Drop your own files here, or let the agent create new ones.

The two starter files below give the AgentLab demo something to look at
on first launch:

- `notes.md`         — a markdown file with a few search hooks
- `calculator.py`    — a small Python module with a typo for the agent to fix
