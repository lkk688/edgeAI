    📘 Jetson + AI — SJSU Edge AI Curriculum¶

    Hands-on labs to learn embedded systems, Linux, AI/LLMs, robotics, and cyber‑AI on the NVIDIA Jetson Orin Nano. Pick your path:

    ▶ Start Here — Lab Slides
    A short, click‑through guide to set up your Jetson and run your first AI model in ~15 minutes. Best for the first lab.
    📚 Full Handbook
    The complete reference — every lab in depth: CUDA, YOLO, LLMs, ROS 2, robotics, security.

    🎞️ Slide decks (short, click‑through — follow along step by step):

    • 🏆 Hackathon — Tracks & Rules — pick a track (LLM · Cyber‑AI · Edge AI) and build your project
    • ▶ Get Started — set up your Jetson and run your first AI model (~15 min)
    • 🌐 Next.js + NVIDIA Nemotron Web App — build & run a streaming chat web app
    • 🤖 ReAct Agents — the reason-and-act loop at the core of every AI agent
    • 🛡️ AI CVE Triage — a tool-using agent that triages security findings

    📖 Want details? Browse the Handbook below — the slides link back to it for deep dives.
